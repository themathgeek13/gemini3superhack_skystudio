## ACT 2: THE SOLUTION (15-50 seconds)

### Scene 3: Drone Fleet Deployment (15-25 sec)
**Camera View**: Free camera (View Mode 1)
**Action**:

- Watch drones deploy and coordinate

- Show them tracking the formation as players move

**Controls**:
1. Click "Deploy Drones" button (or press D)
2. Let drones coordinate for 10 seconds
3. Show them gracefully positioning around the action

**Visual**:
- 12 drones creating a halo around players
- Smooth coordinated movement
- Different heights visible

## TECHNICAL RECORDING NOTES

### Setup
- Browser: Chrome fullscreen at 1920x1080
- Recording: Screen capture at 60 FPS
- Quality: 4K if possible, 1080p minimum
- Lighting: Well-lit room, screen clearly visible

### Pre-Recording Checklist
- [ ] Clear browser cache
- [ ] Close unnecessary tabs
- [ ] Full screen the demo
- [ ] Hide taskbar/dock
- [ ] Test all controls work (arrow keys, buttons, etc)
- [ ] Drones deploy smoothly
- [ ] AI View loads without lag
- [ ] Multi-view renders correctly

### Recording Flow
1. Start recording
2. Let intro play (60 frames of setup)
3. Execute Scene 1-2 (problem)
4. Deploy drones and let play run
5. Switch to multi-view
6. Activate AI View at key moment
7. Demonstrate camera control
8. Return to free view
9. Show view switching montage
10. Final scene setup
11. End screen visible
12. Stop recording

### Expected File Size
- 90 seconds at 1080p 60fps H.264: ~100-150 MB
- 90 seconds at 4K 60fps H.264: ~300-500 MB
